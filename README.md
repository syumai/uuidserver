# uuidserver

* An http server implementation using Go 1.27's standard [uuid](https://pkg.go.dev/uuid) package.

## Demo

### API

* run `uuid.New()`
  - https://uuid.syumai.dev
* run `uuid.NewV4()`
  - https://uuid.syumai.dev/v4
* run `uuid.NewV7()`
  - https://uuid.syumai.dev/v7

### Web (Browser)

* https://uuid-web.syumai.dev
